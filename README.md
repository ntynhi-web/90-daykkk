# 90-Day Life OS

**Nói một phút mỗi ngày. AI biến lời nói thành mục tiêu, milestone, lịch và hành động tiếp theo.**

Ứng dụng tập trung vào ba hành trình chính:

1. Fund & Backtest
2. B2B Marketing
3. Health & Beauty

Mỗi hành trình được chia thành các cột mốc nhỏ, kết nối với công việc ưu tiên, lịch và thói quen hàng ngày.

View your app in AI Studio: https://ai.studio/apps/94b0fd9a-70d4-4f25-966c-1d2b09c6039d

## Chạy trên máy tính

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
